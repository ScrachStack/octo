## Reqirements

- Node.js **v16** or **Higher** is required.
- LEMN Stack
- A LEMN stack is a group of programs that will be the core operators for our website.

L inux

N odeJs
- Nginx
```
sudo apt update
sudo apt install nginx
```
You will be prompted to press Y to install Nginx, do this.
